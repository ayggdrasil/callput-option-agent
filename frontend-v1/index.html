FAQ

Common operator questions
Does this frontend execute transactions directly?

No. Agent runtimes execute through Skill + MCP. Frontend is guidance-only.

Where should private keys be managed?

Inside each external agent runtime environment. Never in frontend state.

Can I skip spread validation?

No. Validation is mandatory before any execution flow.

Is market analysis available in this version?

Not as a template in V1. It is planned for V2.
